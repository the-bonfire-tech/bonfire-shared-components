## Instalação

Clone o repositório:
```bash
git clone https://github.com/thebonfiretech/bonfire-shared-components
```
Entre no diretório do projeto:
```bash
cd bonfire-shared-components
```
Instale as dependências:
```bash
npm install
```

## Uso

```bash
npm run storybook
```
## Build
```bash
npm run build
```
